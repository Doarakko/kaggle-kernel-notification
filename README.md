# kaggle-kernel-notification
## Usage
### 1. Download code
```
$ git clone https://github.com/Doarakko/kaggle-kernel-notification
$ cd kaggle-kernel-notification
```

### 2. Push to heroku
```
$ heroku login
$ git add --all
$ git commit -m 'Yeah!'
$ git push heroku master
```
